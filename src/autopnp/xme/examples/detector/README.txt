========================================
  CHROMOSOME Example "detector"
========================================

Node Name        | Platform
-----------------+----------------
node             | Linux / Windows

Description
-----------
detector shows an example of how to plug and un-plug components during
run-time. Thereby, the component instances receive corresponing
initialization data.
In a cyclic fashion two components are pluged and un-pluged.
